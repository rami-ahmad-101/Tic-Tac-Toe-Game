##Project Description

This project is developed in C# using Windows Forms.  
It simulates a simple and basic XO (Tic-Tac-Toe) game where two players can play against each other on the same computer.

The game focuses on simplicity and clarity, making it suitable for beginners who are learning C# programming and GUI development.

---

###Features

- Create a two-player mode (Player X vs Player O).
- Simple and user-friendly graphical interface.
- Automatic detection of winner or draw.
- Ability to restart the game easily.
- Clear display of turns and results.

---

###Purpose

The main purpose of this project is to practice C# programming concepts, especially:

- Windows Forms (GUI development)
- Event handling
- Basic game logic
- Working with buttons, labels, and forms

---

###Technologies Used

- Language: C#
- Framework: .NET Framework (Windows Forms)
- Concepts: OOP, Event Handling, GUI Design
